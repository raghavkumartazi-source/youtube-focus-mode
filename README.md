# 🎯 YouTube Focus Mode

Transform YouTube into a distraction-free study environment.

YouTube Focus Mode is a modern Chrome extension designed for students, developers, and deep-work sessions. It removes addictive distractions like Shorts and recommendations while adding productivity tools like Pomodoro timer, streak tracking, ambient sounds, and focus analytics.

---

## ✨ Features

### 🧠 Distraction-Free YouTube
- Hide YouTube Shorts completely
- Remove recommended videos sidebar
- Hide homepage feed
- Optional comment section blocker
- Disable autoplay automatically

### ⏳ Productivity Tools
- Built-in Pomodoro focus timer
- Daily focus streak tracking
- XP & study statistics system
- Focus session analytics
- Study goal tracking

### 🎵 Deep Focus Experience
- Ambient background sounds
- Floating draggable focus widget
- Smart notifications & reminders
- Minimal distraction UI

### ⚙️ Customization
- Study Mode ON/OFF toggle
- Whitelist educational channels
- Adjustable ambient sound volume
- Personalized focus settings
- Persistent local storage support

### 🎨 Modern UI
- Dark modern interface
- Smooth animations
- Lightweight & fast performance
- Clean productivity-focused design
